Keymaps
---

General
    ?                    show/hide help view
    Esc / q              go back/exit
    <ctrl+c>             exit immediately

Changes List Pane
    j / ↓                go down
    k / ↑                go up
    g                    go to top
    G                    go to the bottom
    f                    toggle following changes
    <c-r>                reset list
    <space>              toggle watching
    Tab                  switch to diff pane

Diff Pane
    Tab                  switch to changes list pane
    <space>              toggle watching
    <c-r>                reset list

Help Pane
    j / ↓                go down
    k / ↑                go up
